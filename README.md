# Sales Performance Dashboard

## Project Title
Sales Performance Analysis Dashboard built using Power BI to analyze revenue, profit, product performance, and customer behavior across different regions and sales channels.

## Business Problem
Companies generate large volumes of sales data but often struggle to quickly identify profitable products, high-performing regions, and key customers. Without clear visibility into these metrics, it becomes difficult for decision-makers to optimize sales strategies and improve profitability.

## Project Goal
The goal of this dashboard is to transform raw sales data into meaningful insights that help businesses monitor performance, identify growth opportunities, and make data-driven decisions.

## Tech Stack
- Power BI Desktop – Dashboard creation and data visualization
- Power Query – Data cleaning and transformation
- DAX (Data Analysis Expressions) – Calculated measures and KPIs
- Data Modeling – Relationships between tables for interactive filtering
- File Format – `.pbix` project file and `.png` dashboard preview

## Key KPIs
- Total Revenue: 1.24B
- Total Profit: 461.77M
- Profit Margin: 37.36%
- Total Orders: 10.68K
- Total Products Sold: 541K
- Average Order Value: 115.68K
- Average Profit per Order: 43.22K

## Features / Highlights
- Monthly Profit Trend to track profitability over time
- Profit by State to identify high-performing regions
- Sales vs Profit by Product to evaluate product performance
- Profit Distribution by Area Type (City, Town, Village)
- Top Products by Profit ranking
- Top Customers by Revenue analysis
- Revenue comparison across sales channels

## Business Insights
- California generates the highest total profit.
- City locations contribute the largest share of overall profit.
- Sales and profit show a positive relationship across products.
- Wholesale channel generates the highest revenue.
- Profit margins for top products remain around 38–40%.
